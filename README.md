# 32点盤モリス ルール

## 1. 目的
相手の駒を減らし、相手を手詰まりにするか、相手の駒を2枚以下にして勝つ。

## 2. 盤
- 4重の正方形フレームからなる合計32の交点を用いる。
- 各正方形の角と辺の中央に交点がある。
- 同一正方形内では、隣り合う交点が線で結ばれている（角—辺中央—角…）。
- 各辺中央の交点は、隣接する内外の正方形の同じ位置の辺中央と線で結ばれている。
- 斜めの線や中央点はない。
- 駒は線で結ばれた交点間でのみ移動できる。

## 3. 駒と初期設定
- 各プレイヤーは同色の駒を12枚持つ。
- 盤上は空の状態から開始する。
- 先手・後手は任意の方法で決定する。

## 4. 進行
本ゲームは「配置フェーズ」と「移動フェーズ」からなる。

### 4.1 配置フェーズ
- 手番のプレイヤーは、手持ちの駒を1枚、空いている交点に置く。
- 両者の全駒（24枚）が置かれたら移動フェーズへ移行する。

### 4.2 移動フェーズ
- 手番のプレイヤーは、自分の駒1枚を、線で直接結ばれた隣接交点の空所へ移動する。
- 自分の駒が3枚になったプレイヤーは、その手番以降、任意の空いている交点へ移動できる（フライ）。

## 5. ミルと除去
- 自分の駒が一直線に3枚並んだとき（ミルを作ったとき）、相手の駒を1枚盤上から除去する。
- 除去の制限：
  - 原則として、相手のミルを構成している駒は除去できない。
  - 相手の駒がすべてミル内にある場合は、その中から除去してよい。
- 1手で2本以上のミルを同時に成立させても、除去できる駒は1枚のみ。
- ミルを一度崩し、次の手で再び同じミルを作ることは認められる。

## 6. 勝敗
次のいずれかを満たしたとき、直前の手番プレイヤーの勝ち。
- 相手の駒が2枚以下になった。
- 相手が合法手を持たない（手詰まり）。

## 7. 引き分け
- 移動フェーズにおいて同一局面（手番・盤面配置・可動性が同一）が再出現した場合は引き分けとする。
